# Система управления функциями

Клиент-серверная система для управления и выполнения параметризованных функций через HTTP API и TCP CLI.

## Предварительные требования

- Python 3.7+
- Зависимости: `fastapi`, `uvicorn`, `requests`

Установка зависимостей:
```bash
pip install -r requirements.txt
```

## Запуск сервера

Запустите сервер (обрабатывает и HTTP, и TCP):
```bash
python server.py
```
*   HTTP API доступен по адресу `http://127.0.0.1:8000`
*   TCP CLI сервер слушает на `127.0.0.1:8888`
*   Документация API (Swagger UI): `http://127.0.0.1:8000/docs`

## Клиенты

### Демонстрация HTTP клиента
Запускает скрипт, который выполняет CRUD операции и вычисления через REST API.
```bash
python client_http.py
```

### CLI клиент
Подключается к TCP интерфейсу для работы через интерактивное меню.
```bash
python client_cli.py
```

#### Пример ввода для CLI
Скопируйте и вставьте следующие строки по одной для создания и выполнения функции `sum_diff`:
```text
1
sum_diff
x, y
a, b
x + y, x - y
6
sum_diff
10
5
```

## Возможности
- Создание, чтение, обновление, удаление (CRUD) функций.
- Выполнение функций с пользовательскими параметрами.
- Одновременный доступ через HTTP и TCP.
- Движок выполнения на основе `eval()` .
